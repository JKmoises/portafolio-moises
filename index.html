Tecnologías

Html

Maqueto sitios web con HTML de una manera semántica y estructurada siguiendo las buenas prácticas y el estándar para mejor performance y SEO.

Css

Puedo maquetar sitios web estilosos usando características avanzadas de CSS como lo son Flexbox, CSS Grid, Keyframes, Responsive Web Design, etc.

Javascript

Utilizo Vanilla JS para dar interactividad y dinamismo a los sitios web manipulando el DOM, usando eventos, consumiendo APIs, entra otras características.

Git

Manejo el sistema de control de versiones Git para mantener respaldos y distintas versiones de mis repositorios. Tambien lo uso para trabajar en equipo.

Sass

Aplico el preprocesador SASS para codificar CSS de una forma más modularizada, emplear nesting, reutilización más eficiente, entre otras cosas.

Npm

Uso el manejador de paquetes NPM de NodeJS, para así trabajar con dependencias, herramientas o librerías en el desarrollo de mis sitios web.

Webpack

Utilizo el empaquetador de módulos Webpack para minificar código para producción, aligerar imágenes, y usar loaders y plugins útiles.

Mysql

Tengo conocimiento en BD con MySQL para realizar consultas, insertar datos, actualizar datos, eliminar datos y relacionar tablas.
